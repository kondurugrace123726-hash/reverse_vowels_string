# Reverse Vowels of a String 🔤

This project solves the problem "Reverse Vowels of a String" using Python.

## 🚀 Problem Statement

Given a string, reverse only the vowels in the string.

## 🧪 Example

Input:
hello

Output:
holle


## 💡 Approach

- Use two pointers (left & right)
- Move pointers until vowels are found
- Swap vowels
- Continue until pointers meet
